# Transmoggy
Transmoggy is an interface AddOn for Wrath of the Lich King (3.3.5a) that provides an appearance collection, enhanced dressing room features and a transmogrification interface. The transmogrification features rely on a specific transmog implementation and will only work on the Rising Gods server.

Currently, only german and english localization is supported.

## Installation:
- Download the [latest release](https://github.com/hannes121/TransmogAddon/releases/latest)
- Unpack the zip file
- Go into the folder and copy the Transmoggy folder into your WoW/Interface/AddOns folder
